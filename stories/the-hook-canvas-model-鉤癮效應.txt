The Hook canvas model 能幫助新創團隊讓使用者愛上自己的產品。改變人類行為的新創產品越來越多，這些新創產品的未來卻充滿不可預測性，如何在未上市之前就增加成功機率是全世界都在思考的方向。新創公司或許透過快速的 agile 開發直接試水溫、或許用 Lean UX 在開發過程中尋找問題點、或許透過樂觀的凱子創投資金庇護、但最終還是要面對失敗的風險。跟從商業端角度出發的 Businese model canvas 一樣，The Hook canvas model 從使用者端的角度思考，能夠幫助新創團隊優化產品並降低失敗的風險。(只想瞭解 The Hook Canvas 運作方式的朋友可直接跳至下方第2小節)在摩爾定律之下，科技發展速度越來越快，創新的產品越來越多，技術端的問題被解決了，但使用者端的問題日益嚴重，如何改變使用者、教育使用者是新創團隊面臨的一大難題。iphone 之後，矽谷出現前所未有的樂觀投資氛圍，許多目標改造使用者行為的新創產品紛紛出現，但部分的產品中途就退出了賽局、部分產品瘋狂的收買客戶卻沒有證據證明其模式可行(延伸閱讀：O2O 寒冬將至：剩者為王)，甚至連 Apple 推出的最新產品都受到人們質疑” iPad Pro 最大的敌人，可能是纸和筆”，究竟使用者的哪些行為能夠被改變、被教育，現代還沒有一個成熟的研究方法論出現。新的科技出現會產生新的問題解決方法，但改變使用者一個小小的小行為，往往牽扯到整個行為模式的環節，不管是使用者的心理、生理、外在人事時地物影響、其他因素等等，新創產品要成功常需要考慮非常多使用者端的問題。舊有的使用者調查方法，偏向尋找現有的使用者行為證據來推斷新創產品是否會成功，但能夠推斷的準確依然有限，且越是改變使用者行為的情況越就難以預測，若又當牽扯到付費方式、信任、使用倦怠期、進入障礙、網路效應後，研究、預測新產品的未來更是難上加難。即使有前述 iphone 成功的案例，矽谷還是堆了滿山遍野失敗的 Startups，過往的使用者研究方法已經追上新時代的步伐，連設計大師 唐·諾曼都認為教育跟設計研究方法已經跟不上時代，不管新的設計研究方法論何時誕生，降低失敗風險的 model 相繼出現來解決現有問題，這其中之一，就是 Hook Canvas Model。UX 並非新創團隊的唯一解，但是個很好的思考方式，切勿凡事以 UX 為唯一思考依據，新創團隊決策時要考慮更廣更闊的格局。UX 的面向十分廣闊，在此不做太多的說明，但其中很重要的兩點：UX 思維重視使用者的使用行為、UX 思維除了考慮使用者和產品的互動外也會考慮整個產品服務的全部過程。透過 UX 思維更能了解、提供使用者願意接納的產品。而 Hook Canvas Model 就是 UX 思維下可應用的工具之一。The Hook Canvas 是 Nir Eyal 在大約 2013 整理完成的理論，操作概念分為四個步驟，分別是：Trigger、Action、Variable Reward、Investment。其應用概念為幫助使用者解決痛苦的問題後給予使用者快樂，讓使用者願意一再重複使用該產品，最終為產品帶來穩定的使用客戶。可翻譯為”觸發源”，也就是想要使用產品的動機，Trigger 又被 Nir Eyal 分為 External 和 internal：以製作一份學校課程報告的過程為例：外部 Trigger 就像是，當我們發現資料不夠完整時我們會使用 Google搜尋 。內部 Trigger 則會是，當我們對於報告中論點感到不安時我們會想用 Google 搜尋做確認 這概念也可以用台灣所說的需求、問題點，中國的痛點來理解， Nir Eyal 認為唯有幫助使用者解決問題，方能讓使用者願意開始使用產品。Nir Eyal 對 Action＝行為的定義非常廣，是為”最簡單就可以獲得快樂的行為”。 小弟我揉合 MVP 的核心價值概念，定義為”做了就能得到快樂或解除痛苦的關鍵行為”，在產品的呈現中，可能是個服務，可能是個功能也有可能是一整個完整的流程。這一步牽扯到說服設計和進入障礙的相關理論，使用者往往連行動都沒有就放棄使用產品了，例如：連註冊都不願意就放棄使用某些軟體的客群。Reward，獎勵，一個讓人能感到快樂的回饋，呈現的方式非常多種，有時解決使用者的問題就等同獎勵，完成 Action 後不用多餘動作就能達成；有時需透過行銷的手法讓使用者瞭解自己行為的價值；有時可透過 Gamification 的概念給予使用者愉悅感(例如遊戲中戰鬥結束的勝利畫面)，同時可配合服務設計概念中的”實體證據”，故意將獎勵特別呈現給使用者，滿足其心理需求。Nir Eya 特別強調 Variable 變動的概念，Nir Eya 認為變動獎勵才能更有效的促進使用者培養習慣，就像是賭博上癮的心裡、最近手機遊戲最夯的隨機抽卡機制。即使該項步驟增為 Variable Reward，但許多新創產品確實沒有 Variable 也能達到改變使用者行為的效果。Nir Eya 稱 Investment 為”投資”，使用者會為了想獲得的獎勵進行投資＝使用，投資的動作會導致使用者繼續連接到新的 trigger。Investment

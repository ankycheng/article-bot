在社群中陸續看到有人開始試用了，而我的信終於也在 4/4 寄過來啦～Invision Studio 是一個新的介面設計工具，Invision 公司旗下有廣為人知的原型工具 Invision App、以一擋百的 Sketch 外掛 Craft ；所以我對 Invision Studio 抱持著非常大的期待，官網透露他們未來會出 PC 版，我認為此舉將大幅減少 UI 設計的門檻，讓學生族群或是有興趣的普羅大眾皆可學習使用，看得出 Invision 的野心那麼，以下都是超主觀的個人看法！4. 提供黑白兩色的主題做切換5. 可以開 Sketch 檔！！！！太神啦！！！！！2. 個人覺得 App 的操作有點怪，Stop Mirroring 的選項很快就消失了，每次我想結束都不知道從哪裡離開3. 有時後按住 Option 看距離的時候計算的反應會慢半拍，不過這是偶發狀況，所以沒有截圖證明 …像是 Symbol （在 Invision 叫做 Component ）、Library 功能其實都大同小異這邊就不多贅述了個人蠻喜歡 Sketch 的 Prototyping 功能，簡單直覺好用；Sketch 目前只支援四種基礎的轉場效果，我覺得它 Plugin 和工具這麼多了其實這樣就夠；然而 Invision Studio 提供了多樣化的動態效果…只要照著範例做就可以實現華麗的翻頁動畫，檔名要一樣；類似的工具可以參考我之前寫的設計工具： Flow 介紹文而且只要簡單兩步驟：在右側欄位的 Scrolling 選 Vertical、再把 Header Pin 起來（固定不動），就這樣簡簡單單的搞定滾動動畫但是！官網上的這個超神介面卻遲遲沒有出現，遍尋不著，有點失望；這會包在 Invision Studio 裡還是會成為 Plugin？更新：感謝張尊尼網友，這個酷炫的功能放在有點隱密的地方，在做介面串接時按下最下方的 Edit Timeline裡面長這個樣子，左邊可以自行微調時間的長度和先後、右邊可以預覽效果左上角也有選項可以提供轉場效果，我覺得這樣的功能基本上已經可以應付大部分的動態了，不要太複雜的應該都做得出來（或是示意得出來）即使還未正式上線、或許還有一點點瑕疵，但在粗略的玩過一輪 Invision Studio 之後，對他的興趣有增不減；這可能是一個怪物級的產品！Sketch 真的要小心啦～（快把 App 配對的問題修好啦！）但是對我來說 Sketch 的 Plugin 非常多，很多服務比如說切圖工具、版本控制工具、一次命名的工具、自動產生圖文的工具、自動灑小花花的工具…要割捨實在需要一些勇氣（和陣痛期）；這點只能再觀望看看未來 Invision Studio 有多少 Plugin 或新功能了以上～這是我對 Invision Studio 的心得，每個人在意的點不同，還是建議有空的話親身體驗看看，還沒登記的同學趕快去官網登記喔～From a quick cheer to a standing ovation, clap to show how much you enjoyed this story.設計工具分享 / 學習筆記 / 個人心得AAPD專注於分享數位產品設計的相關資訊，並且致力在平台上創造更多的交流與互動，我們關注UI設計、UX設計、設計師的個人成長、設計趨勢與產業動態等，希望透過這些知識的傳遞，能夠降低每位設計師成長的過程中所遇到的阻礙。歡迎來信投稿：aapdgo@gmail.com

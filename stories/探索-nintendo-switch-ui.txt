從去年 7 月入手 Nintendo Switch （之後文內統稱 NS）之後，在使用的同時，也一直去觀察跟思考 NS 的 User Interface Design。就一直想發表相關 UI 的文章。首先 Nintendo 這次的主機同時是掌上型遊戲機跟電視遊樂器。所以 Nintendo 官方也一直主打著 NS 的方便攜帶性。要便利於攜帶的話，體積大小就很重要，從圖中可以看到，長度跟 A5 筆記本的比較，以及厚度的比較。既然是打著方便玩家攜帶遊玩的主機，啟動速度就是很重要的一點，而 Nintendo 有注意到這點，所以有快速啟動 NS 的方式，其中一個是按下電源鍵，另一個是按下右邊手把的 home 鍵，影片是按下右邊手把的 home 鍵快速喚醒，從 sleep mode 到啟動遊玩，不到幾秒的時間就完成了，蠻佩服 Nintendo 這點真的做得不錯。這種啟動速度讓使用者在遊玩 NS 的時候，要移動至不同目的地時，真的很方便。進入系統的首頁，我們可以清楚的看到 NS 的 UI，主要分為幾個層面，有主要列表、次要列表、系統設定相關。首先是佔據整個畫面最多的主要列表，以水平的正方形方式呈現。看得出來四方形的呈現，可以讓遊戲公司在遊戲主畫面有很好的編排。有個小地方值得注意的是，NS 有分有實體的遊戲片跟數位版的遊戲，在下圖也可以看到不同的 icon 來提示。次要列表可以看出是很明顯的比遊戲列表弱化很多，次要列表的目的，是提供功能的導航。而且第一個紅色的 News 有新的發表時，會有一顆綠點提示。前三個的顏色比較不同，可以猜想應該是 Nintendo 強調的功能，所以在顏色安排上有比較特別。左上角是個人設定，有朋友在線上的時候，個人設定的上方會出現數字。系統設定相關的層級就是最弱的呈現，這邊就是系統相關資訊跟操作提示的，如圖中我們可以看到時間、電量、按鈕的提示，以及左下角是機體模式的提示。如前面所說 NS 是可以攜帶的遊戲主機，也可以接著電視遊玩，所以左下角就是在提示，目前 NS 在哪一種模式運作。在手持模式下，啟動 NS 會有一個螢幕鎖，雖然螢幕顯示 A 按鈕為 Start，但是在這個時候其實按下任意的按鈕可以進入下一個解鎖的畫面。解鎖的方式就是同一顆按鈕連續按三下，就可以解除鎖定了。不過個人把這個動作當作是遊戲遊玩前的小指令，就上是進入駕駛倉要啟動機器一樣的感覺。此外解鎖後進入的畫面會停在上一次遊玩的畫面上，這個部分在之前快速開機的影片中可以看到。個人覺得這點體驗者的很好，不用每次重新啟動遊戲。當你從遊戲回到主畫面，尚未結束遊戲時，可以看到目前是那個帳號正在遊玩遊戲，而且 avatar 部分會一直彈跳，再搭配鮮豔的 focus 狀態，就像霓虹燈一樣，感覺就像是在告訴 User 趕快回來玩，充滿著很好玩的氛圍。下方影片將演示，將 NS 的 Joy-con 手把抽離 NS 本體，以及跟 NS結合的動畫互動。從影片中可以看到當紅色 Joy-con 手把抽離 NS 本體時，銀幕畫面左側也做出紅色框線往上離開的動畫，抽離之後同時在螢幕左下角有很清楚提示目前狀態，換另一邊藍色 Joy-con 手把的時候，就變成藍色的框線，經實驗之後，那個框線是跟著目前的 Joy-con 手把顏色變化，也許是這樣的微不足道小細節上的處理，讓人覺得 Nintendo 在設計遊戲主機的時候，有很用心在處理相關互動的小細節上面。NS 是家用跟掌上兼容的遊戲主機，提供了玩家三種遊玩方式，第一種是 TV mode，第二種是 Tabletop mode，第三種是 Handle mode，所以 NS 的 Joy-con 控制器的設計就是對稱的，方便我們可以直接跟朋友一同遊玩。在 Handle mode 中，主機的螢幕也是可以觸碰進行互動。Joy-con 的配對只要將他們連接到 NS 的主機上方，Pro 控制器則是通過 USB 連接上 NS 的底座來配對。在切換主機或 Joy-con 模式時，要求玩家同時按下 L ＆ R 按鈕來啟動 Joy-con ，來確定玩家正在使用的控制器跟使用的方向。這是一個很不錯的互動，因為 Joy-con 可以有兩種不同的手持模式，這種方式能很快幫助系統辨識玩家是如何拿著 Joy-con 。NS 也提供玩家登入不同的遊戲帳號，我們可以從主畫面的左上角進入使用者資訊的頁面。NS 在啟動遊戲時，也會詢問玩家要使用哪個帳號來啟動遊戲，這看起來像一個很好的功能，不過玩家會在玩多個遊戲時，每次都詢問玩家一次帳號，反而有點多餘，能夠紀錄目前登入使用的帳號，會是一個好的結局方法。NS 目前只提供了兩種主題，一個是 dark theme 另一個是 light theme 個人推薦 dark theme，因為 light theme 看久了會覺得對眼睛有點壓迫感，在 TV mode 會更加的明顯，不過 NS 是新的遊戲主機，說不定將來 Nintendo 會開始推出不同的 theme 來給玩家有更多的選擇。隨著使用 NS 的時間越來越長，會覺得 Nintendo 在一些小細節上的處理，真的蠻用心的。不管在主機的介面上，或者是在遊戲的體驗上，常常會有一些小互動細節藏在裡面，等察覺到時，會驚嘆這個小地方居然也做到互動，也太細了吧。最後 NS 是一台全新概念的遊戲主機，或許在不少地方還是有改進的部分，不過很期待 Nintendo 在未來的創新跟改進，尤其最近的 Nintendo Labo(ニンテンドーラボ) 就讓人覺得很期待。另外我的 NS friend code 是 SW-6023-2325-0928，歡迎大家加好友一起玩。From a quick cheer to a standing ovation, clap to show

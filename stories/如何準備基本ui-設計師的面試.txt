大家好，我是羅伯特。每到年關前有些人總會想要換個環境，不管是對於原本工作內容不滿意或是遇到瓶頸想要成長的設計師們，能夠知道自己下一個人生階段去做努力去奮鬥是非常好的事情。而會想打這篇文章的原因，是因為小弟目前任職的公司開出了一個 “Senior UI Designer” 的職缺，雖然給的薪水不到優渥，但還算符合開出條件的水準，內容如下：【工作職缺】網頁/手機使用者介面設計師Web/App UI Designer【負責的工作內容】 1. 實際參與公司主要產品 Web／App的 UI 設計2. 與 UX Designer 溝通，提出最佳的介面設計，兼具美感與易用性3. 與工程團隊密切溝通和調整，在工程可實作的範圍中提供使用者最佳體驗4. 遵循並改善本公司的設計規範5. 沒事的時候可以畫一些可愛頭像或是插圖調劑身心來投履歷的朋友從2月初到現在大約有百來封，可是你猜猜我們實際上約面試的人有幾位？你可以質疑問說：「你是不是隨便看看？」來信的朋友我可以掛擔保，你們的作品我都有過目，我也可以立刻說出哪裡不適合，也因為不適合的程度高到有點嚇人，不用說稍微資深的介面設計師，可能連新人的介面設計師職位都應徵不上，如果你不知道自己哪裡需要調整，可以將自己準備的內容檢視以下幾點，希望對你有幫助。2.另外還有落落長的履歷…努力的跟我講你的家庭狀況、平常興趣、多久去運動一次、聽什麼音樂，都跟我無關啊！不是說履歷不能寫，你可以簡單描述之前跟應徵此行業有關的經歷，讓我更加了解為什麼你要來投這個工作。3.作品集內容雜亂無序…最讓人不能接受的就是直接將一堆作品丟在雲端上要面試官去看，如果你連自己的成品都整理不好了，那為何你敢要求別人看那一堆資料夾？另一種雜亂是大雜燴，什麼東西都有，平面、攝影、廣告、企劃、素描等，我可以理解你可能沒什麼做app介面的經驗，所以想用平面設計或者其他作品來表現自己的能力，可是一口氣放了三十幾張廣告設計…幾本上要求在求職信中附上方便點閱觀看的作品集、簡單描述自己相關的工作經歷即可以，作品不用多可以表達出自己能力最重要。這樣其實就是達到基本要求了。3. 不要放上有趴數能力的bar，例如英文90％、sketch70%、phtotshop87%…到底從何比較而來？？？能夠掌握並且運用在工作上就寫上去，如果還是想描述熟稔狀況，建議可以寫「普通」、「尚可」等，會比87%的絕對值好。如果以上都做到，作品能力也符合公司需求的話，應該就可以順利拿到面試資格，最後提幾項面試時需要注意的事情。「好…這一頁是我設計的介面，左上有返回鍵、右邊有搜尋、因為老闆說要綠色所以放了綠色，下面這一排是表單可以點進去看資料，旁邊有照片，大概是這樣」恩…請問我看起來是看不見的朋友嗎，希望你解釋就是想知道你的想法。「好…這一頁是我研究使用者習慣後跟ux 設計師討論出來的結果，左上有返回鍵符合iOS的規範、因為這裡資料量非常大所以提供搜尋、這一頁希望不要給使用者過多的心理壓力所以使用綠色，表單的距離有調整過，就算內容過多也不會影響閱讀，旁邊照片讓使用者更確定內容是什麼，大概是這樣」對於同一個畫面，如果在設計的時候有考慮周全產生這樣的描述應該不困難才是。說了那麼多，也有那麼多相關設計前輩提供的意見，但願各位在推薦自己的同時審視一下這幾點，相信你可以更順利找到心目中理想的工作，如果還有相關的問題或者對於我提供的建議有別的看法，歡迎跟我討論！希望這些拙見能夠幫助提供你們一些方向與檢視方法。From a quick cheer to a standing ovation, clap to show how much you enjoyed this story.人稱板橋瑟夫·高登-李維（誰？） UI designer, Taiwanese.AAPD專注於分享數位產品設計的相關資訊，並且致力在平台上創造更多的交流與互動，我們關注UI設計、UX設計、設計師的個人成長、設計趨勢與產業動態等，希望透過這些知識的傳遞，能夠降低每位設計師成長的過程中所遇到的阻礙。歡迎來信投稿：aapdgo@gmail.com
